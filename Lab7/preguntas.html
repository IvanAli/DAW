Preguntas

¿Qué hace la función phpinfo()? Describe y discute 3 datos que llamen tu atención.

Muestra una gran cantidad de información del estado actual de PHP, información como las opciones de compilación, la versión de PHP, el ambiente de PHP, la versión del sistema operativo, la licencia PHP, etc.

Parámetro what: Configura el output que la función dará, dependiendo de qué constante se pase en este parámetro. Por ejemplo: INFO_GENERAL, INFO_CREDITS_ INFO_ALL.

La función regresa un valor bool, que representa el éxito de la función.

Funciona como herramienta para debuggeo, ya que contiene datos de servidor, cookies, ambiente, GET y POST.

¿Qué cambios tendrías que hacer en la configuración del servidor para que pudiera ser apto en un ambiente de producción?

Se tienen que agregar las direcciones relativas de las carpetas que contienen archivos que requieren de la aplicación web para funcionar, además de la que por defecto ya está (htdocs) en caso de que haya archivos fuera de esta.

¿Cómo es que si el código está en un archivo con código html que se despliega del lado del cliente, se ejecuta del lado del servidor? Explica.

El cliente envía una petición a través de una URL al servidor, y el servidor responde con un archivo html que se crea dinámicamente del lado del servidor.
